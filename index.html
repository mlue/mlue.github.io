Irc daily log use visulaization

Timeline

The bottom blue graph is a chronological graph of the total # of lines typed in the chatroom that day. You can drag and select a fraction of the timeline in order to make the other graphs reflect data collected only over that period of time.

User Bar Chart

The graph with the colored blocks with names on them is a view how much users chatted relative to one another. As in the case of the last graph, you can select one user or more users here in order to have the Timeline graph show data pertaining to the selected users.

Day Pie Chart && Time of Day Chart

You might need to widen your window to see these on the far right. It tried to fit too much information on one screen. This one reflects the relative number of lines typed per day of the week and the time of day typed respectively.
